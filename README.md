# PirMod (Lethal Company)

A simple Lethal Company mod that adds fun mechanics and quality-of-life improvements. 

## Features

* **Infinite Sprint:** You never run out of stamina. Run forever!
* **Turret Breaker:** You can now destroy turrets by hitting them with a Shovel or Stop Sign.
* **Mine Traps:** Enemies now trigger landmines. If a monster steps on one, it explodes and dies.
* **Starter Kit:** A Shovel automatically spawns in the ship every time you land.

## Installation

1.  Download and install **BepInEx** (if you haven't already).
2.  Download the latest release of this mod.
3.  Place `PirMod.dll` into your `Lethal Company/BepInEx/plugins/` folder.
4.  Run the game!

## Requirements

* Lethal Company
* BepInEx 5.4.21+

## For Developers
